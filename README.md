Name:Employee Management System
Project Description:

A simple object‑oriented JavaScript project that models employees, managers, and company structure using ES6 classes. The system demonstrates inheritance, method overriding, and basic data organization through class instances.

Technology  Used:

HTML

JavaScript 

CSS

How to run:

Downloads or navigate to the repository

Open the project folder

Launch the index file on a browser

Navigate to the console

View the output generated by the cc_6.js file

What I learned:

How to use the class and console syntax to structure data

How to use inherent data in JavaScript using extend and super()

How to override parent classes using describe()

How to orginie many objects in a parent class
